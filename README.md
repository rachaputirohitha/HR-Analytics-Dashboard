# HR Analytics Dashboard
## Project Overview
  Developed an interactive HR Analytics Dashboard in Power BI to analyze employee attrition and workforce demographics. The dashboard helps HR teams identify attrition trends across departments, gender, age groups, education fields, and job satisfaction levels.
## Objective
To provide actionable insights that support employee retention strategies and improve workforce planning through data-driven decision-making.
## Key Metrics Analyzed
* Overall Attrition Rate
* Attrition by Gender
* Department-wise Attrition
* Employee Distribution by Age Group
* Job Satisfaction Analysis
* Education Field-wise Attrition
* Attrition by Gender and Age Group
## Tools Used
* Power BI
* Excel
* DAX
